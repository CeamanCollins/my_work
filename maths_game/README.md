# Maths Game

## Overview

This [**Maths Game**](./maths_game.py) script is a simple and interactive Python program designed to help users practice and improve their basic math skills. It offers various game modes to make learning fun and engaging.

---

## About

This project was created as part of the *Python Programming* course at UCD Professional Academy. The course emphasized building practical Python applications for real-world scenarios, and this project demonstrates the application of core Python programming concepts.

---

## Features

- Multiple game modes, including:
  - Addition practice
  - Subtraction practice
  - Multiplication practice
  - Mixed mode (randomized math problems)
- Tracks the user's progress while playing.
- Designed to be beginner-friendly and intuitive.

---

## How to Run

### Running in GitHub CodeSpaces:
1. Open the repository in [GitHub CodeSpaces](https://github.com/features/codespaces).
2. Navigate to the **maths_game** folder.
3. Execute the program by running:
   ```bash
   python mathsgame.py
   ```

### Running Locally:
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/CeamanCollins/my_work.git
   cd my_work/maths_game
   ```
2. Ensure Python is installed on your machine.
3. Run the program:
   ```bash
   python mathsgame.py
   ```
   On Windows:
   ```bash
   [path-to-python]\python.exe mathsgame.py
   ```

This script requires **no external dependencies** and relies entirely on Python's standard library. Ensure you have Python 3.6 or higher installed.
